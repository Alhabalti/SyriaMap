# SyriaMap

واجهة تفاعلية لعرض خريطة المحافظات السورية مع معلومات أساسية، وتصفح بصري لطبقات الخريطة والبيانات المرتبطة بها.

## المتطلبات

- Node.js 18 أو أحدث

## التشغيل محليًا

1. ثبّت الحزم:
   ```bash
   npm install
   ```
2. شغّل المشروع:
   ```bash
   npm run dev
   ```
3. افتح الرابط الذي يظهر في الطرفية، وغالبًا يكون:
   ```
   http://localhost:3000
   ```

## البناء للنشر

لإنتاج نسخة جاهزة للرفع على GitHub أو للاستضافة:

```bash
npm run build
```

يتم الآن توليد مسارات الملفات بشكل نسبي افتراضيًا حتى يعمل الموقع بشكل صحيح على أغلب منصات الاستضافة. وإذا احتجت مسارًا مخصصًا، أضف المتغير التالي قبل البناء:

```env
VITE_BASE_PATH=/your-base-path/
```

## إعدادات اختيارية

إذا كنت تستخدم ميزات Gemini الموجودة في المشروع، أنشئ ملف `.env.local` وأضف:

```env
GEMINI_API_KEY=your_api_key_here
```

يمكنك أيضًا استخدام [.env.example](.env.example) كنموذج.

## ملاحظات

- المشروع مبني بـ Vite + React + TypeScript.
- ملفات البيانات الأساسية موجودة في `src/data`.
